# Project RANGMANCH

**RANGMANCH** is an AI-native entertainment companion that transforms a user’s real-life moods, moments, and social context into personalized interactive episodes, social mini-dramas, and evolving story worlds.

## Project vision
RANGMANCH is not a generic chatbot, not a therapist-first AI companion, and not a copy of existing character-chat products. It is a companion-led personal entertainment operating system designed to turn everyday life into a living entertainment universe.
