# Submission Workspace

## Purpose
This folder contains the final packaging workspace for Project RANGMANCH.

It should eventually contain:
- final deck PDF
- final APK
- screenshots
- appendix materials
- backup assets for review and presentation

## Intended structure
- `deck/` → final deck files
- `apk/` → final APK and build notes
- `screenshots/` → polished prototype screenshots
- `appendix/` → references, architecture notes, extra evidence
- `backups/` → backup visuals, backup narration notes, alternate screenshots

## Packaging principle
Only polished, intentionally named, reviewer-ready assets should go into this folder.
