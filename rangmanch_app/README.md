# RANGMANCH Prototype App

This package contains the Flutter prototype shell for Project RANGMANCH.

## Purpose
The `rangmanch_app` package is responsible for implementing:
- mobile routes and screens
- design system primitives
- shared UI components
- schema-based rendering surfaces
- mock-backed generation flows
- local continuity storage abstraction
- demo-first flows for the case study

## What this package is not yet
This package is not yet:
- the fully integrated production backend app
- the final moderation system
- the final analytics system
- or the final AI inference layer

It is the prototype architecture and MVP app shell.

## Tech stack
- Flutter
- Dart
- flutter_riverpod
- go_router
- shared_preferences
- google_fonts
- flutter_animate
