# Documentation Index

This folder contains the full documentation stack for Project RANGMANCH.

## Documentation philosophy
RANGMANCH is being built documentation-first. The project is intentionally grounded in clear strategic reasoning before product design, AI system design, UI design, prototype development, and pitch creation.

## Folder map

### `phase-00-strategic-grounding/`
Contains all Phase 0 documents used to lock:
- product meaning
- strategic intent
- category definition
- boundaries
- metaphor
- principles
- MVP wedge
- success criteria
- exit gate

### `references/`
Contains source notes and brief references.
