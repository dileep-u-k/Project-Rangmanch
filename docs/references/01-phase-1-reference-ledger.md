# 01 — Phase 1 Reference Ledger

## Purpose of this document
This document is the master ledger of all external references used during Phase 1.

Its purpose is to:
- maintain a clean evidence trail
- support the brief's requirement to provide references used during understanding and ideation
- make later synthesis and pitch work easier
- reduce citation loss across the project

## Reference logging format
For every source, capture:
- Source ID
- Title
- Publisher / Author
- Year / Date
- Research pillar
- Key takeaway
- How it affects RANGMANCH
