# 04 — Onboarding Screen Spec

## Purpose of this document
This document defines the onboarding experience of Project RANGMANCH.

Its purpose is to specify:
- what the user sees first
- what minimal inputs are collected
- how the concept is communicated quickly
- and how onboarding transitions into the first wow path

---

## Onboarding design goals
The onboarding should feel:
- short
- cinematic
- alive
- companion-led
- low-friction
- immediately differentiated from a generic AI app

---

# Required onboarding structure

## Screen 1 — World entry
Purpose:
- establish mood
- establish the companion
- establish that this is a living entertainment universe

Must communicate:
- this is not a chatbot
- this is a playful companion-led world
- your daily life can become entertainment here

---

## Screen 2 — Identity / vibe setup
Purpose:
- capture nickname
- capture a lightweight vibe signal

Input examples:
- nickname
- current vibe
- optional quick mood

---

## Screen 3 — Style selection
Purpose:
- collect humor flavor
- collect companion tone preference

Input examples:
- witty
- dramatic
- chaotic
- chill
- savage-light

---

## Screen 4 — First episode trigger
Purpose:
- transition directly into the product
- give a strong CTA into Home or Episode

CTA examples:
- “Enter my universe”
- “Start today’s act”
- “Let’s make this dramatic”

---

# MVP implementation guidance
For the MVP prototype, onboarding can be compressed into a **single strong screen** with:
- world intro copy
- nickname field
- vibe chips
- humor style chips
- enter button

This is acceptable if it remains:
- polished
- legible
- and compelling

---

# Visual guidance
The onboarding should feel:
- dark-stage aesthetic
- immersive but clean
- strong title typography
- obvious primary CTA
- not form-heavy

---

# Interaction guidance
The onboarding should minimize typing.
Use:
- chips
- segmented choices
- one text input max

---

# Success criteria
A successful onboarding screen makes the user feel:
- intrigued
- oriented
- lightly personalized
- and ready to begin immediately

---

## Final summary
RANGMANCH onboarding should be a **world-entry moment**, not an account-setup chore.

Its job is to create:
- identity
- tone
- emotional energy
- and momentum into the first wow flow.
